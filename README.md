# Headhunter in Vue.js

This is a simple Vue.js project for a headhunter application.

## Features

- Navigation bar with links to different pages.
- Pages include Home, Sign up, and Login.
- Each page displays corresponding content.

## Technologies Used

- Vue.js
- Bootstrap 5

## How to Run

1. Clone this repository.
2. Open the `index.html` file in your web browser.

## Project Structure

- `index.html`: Main HTML file containing the Vue.js code.
- `script.js`: Vue.js script for handling data and rendering.
- `styles.css`: Custom CSS styles for the project.
